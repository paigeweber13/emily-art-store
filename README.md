# Emily's Art Store
A store and portfolio for Emily Coulter to showcase her art. **Releases Every
Sunday**

# Dependencies:
 * Ruby 2.6.0
 * Devise secret key set in the environment variable
   "DEVISE_TOKEN_AUTH_SECRET_KEY"
